# MiniGame

4가지 미니게임이 가능한 안드로이드 앱





## 기능

1. 두더지잡기 게임

   제한시간 안에 최대한 많은 두더지를 터치해야 합니다.

2. 머핀 게임

   무작위로 나오는 머핀과 똥을 빠르게 분류해야 합니다.

3. 카드 뒤집기

   제한시간 안에 최대한 많은 카드의 짝을 맞춰야 합니다.

4. 바위 깨기

   제한시간 안에 최대한 빠르게 터치하여 바위를 깨야 합니다.
